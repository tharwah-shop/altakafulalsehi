# تحديث القائمة الديناميكية للمصادقة

## 📋 نظرة عامة

تم تطوير نظام قائمة ديناميكية في شريط التنقل تتغير حسب حالة تسجيل الدخول للمستخدم.

## ✨ المميزات الجديدة

### 1. القائمة للمستخدمين غير المسجلين
- **زر دخول**: يوجه إلى صفحة تسجيل الدخول
- **زر اشترك الآن**: يوجه إلى صفحة الاشتراك

### 2. القائمة للمستخدمين المسجلين
- **قائمة منسدلة باسم المستخدم** تحتوي على:
  - **حسابي**: يوجه إلى لوحة التحكم الشخصية
  - **الملف الشخصي**: لتعديل البيانات الشخصية
  - **اشتراكاتي**: لعرض وإدارة الاشتراكات
  - **تسجيل الخروج**: لتسجيل الخروج من النظام
- **زر اشترك الآن**: يبقى متاحاً للاشتراكات الجديدة

### 3. التوافق مع الهواتف المحمولة
- **شريط التنقل السفلي** يتغير أيضاً حسب حالة تسجيل الدخول
- **قائمة منسدلة للأعلى** في الهواتف للمستخدمين المسجلين

## 🛠️ الملفات المحدثة

### 1. Layout الرئيسي
- **الملف**: `resources/views/layouts/frontend.blade.php`
- **التحديثات**:
  - إضافة منطق `@auth` و `@endauth`
  - قائمة منسدلة للمستخدمين المسجلين
  - تحديث شريط التنقل للهواتف
  - إضافة CSS مخصص للقوائم المنسدلة

### 2. المسارات الجديدة
- **الملف**: `routes/web.php`
- **المسارات المضافة**:
  - `/dashboard` - لوحة التحكم الشخصية
  - `/profile` - الملف الشخصي
  - `/my-subscriptions` - اشتراكات المستخدم

### 3. الكنترولر الجديد
- **الملف**: `app/Http/Controllers/UserDashboardController.php`
- **الوظائف**:
  - `index()` - عرض لوحة التحكم
  - `profile()` - عرض الملف الشخصي
  - `updateProfile()` - تحديث بيانات المستخدم
  - `subscriptions()` - عرض الاشتراكات

### 4. صفحات العرض الجديدة
- **`resources/views/user/dashboard.blade.php`** - لوحة التحكم الشخصية
- **`resources/views/user/profile.blade.php`** - صفحة الملف الشخصي
- **`resources/views/user/subscriptions.blade.php`** - صفحة الاشتراكات

2. **تسجيل الدخول** - استخدام البيانات التجريبية

5. **تسجيل الخروج** - التأكد من العودة للحالة الأولى

# نظام استيراد وتصدير المراكز الطبية

## نظرة عامة

تم إضافة نظام شامل لاستيراد وتصدير المراكز الطبية يدعم ملفات Excel و CSV، مما يسمح بإدارة البيانات بكفاءة عالية.

## الميزات

### 1. تصدير البيانات
- تصدير جميع المراكز الطبية إلى ملف Excel
- تنسيق احترافي مع عناوين باللغة العربية
- تضمين جميع البيانات المهمة
- تنسيق تلقائي للأعمدة

### 2. استيراد البيانات
- استيراد من ملفات Excel (.xlsx, .xls) و CSV
- التحقق من صحة البيانات
- معالجة الأخطاء مع تفاصيل واضحة
- دعم التحديث والإنشاء الجديد

### 3. قالب Excel
- قالب جاهز للتحميل
- أمثلة على البيانات الصحيحة
- تعليمات واضحة

## كيفية الاستخدام

### التصدير

1. انتقل إلى صفحة إدارة المراكز الطبية
2. اضغط على "استيراد/تصدير" → "تصدير إلى Excel"
3. سيتم تحميل ملف Excel يحتوي على جميع البيانات

### الاستيراد

1. انتقل إلى صفحة إدارة المراكز الطبية
2. اضغط على "استيراد/تصدير" → "استيراد من Excel"
3. حمل ملف Excel أو CSV
4. اضغط "استيراد البيانات"

### تحميل القالب

1. انتقل إلى صفحة الاستيراد
2. اضغط على "تحميل قالب Excel"
3. املأ البيانات في القالب
4. ارفع الملف للاستيراد

## حالات المركز

- `active` - نشط
- `inactive` - غير نشط
- `pending` - معلق
- `suspended` - موقوف

## حالات العقد

- `active` - نشط
- `expired` - منتهي
- `pending` - قيد المراجعة
- `suspended` - معلق
- `terminated` - ملغي

## معالجة الأخطاء

النظام يتعامل مع الأخطاء بذكاء:

1. **أخطاء التحقق**: يتم عرض تفاصيل الأخطاء مع رقم الصف
2. **البيانات المكررة**: يتم تحديث البيانات الموجودة
3. **البيانات الناقصة**: يتم تخطي الصفوف غير المكتملة مع تسجيل الخطأ

## الملفات المضافة

### Controllers
- `app/Http/Controllers/Admin/MedicalCenterController.php` - إضافة وظائف الاستيراد والتصدير

### Classes
- `app/Exports/MedicalCentersExport.php` - كلاس التصدير
- `app/Imports/MedicalCentersImport.php` - كلاس الاستيراد

### Views
- `resources/views/admin/medical-centers/import.blade.php` - صفحة الاستيراد
- تحديث `resources/views/admin/medical-centers/index.blade.php` - إضافة أزرار الاستيراد والتصدير

### Routes
- إضافة روتات الاستيراد والتصدير في `routes/web.php`

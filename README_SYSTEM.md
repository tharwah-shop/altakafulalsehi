# نظام التكافل الصحي - Laravel

## نظرة عامة
نظام شامل لإدارة المراكز الطبية والمنشورات مع نظام صلاحيات متقدم وتصنيفات جغرافية.

## المميزات الرئيسية

### 1. نظام الصلاحيات والأدوار
- **الأدوار المتاحة:**
  - مدير عام (super_admin): جميع الصلاحيات
  - مدير (admin): معظم الصلاحيات
  - مدير مركز طبي (medical_center_manager): إدارة مركزه ومنشوراته
  - مدير المحتوى (content_manager): إدارة المنشورات والتقييمات
  - مستخدم عادي (user): صلاحيات محدودة

- **الصلاحيات المتاحة:**
  - إدارة المستخدمين والأدوار
  - إدارة المراكز الطبية
  - إدارة المنشورات
  - إدارة التقييمات
  - الوصول للوحة التحكم

### 2. نظام المراكز الطبية
- **أنواع المراكز:**
  - مستشفيات (hospital)
  - عيادات (clinic)
  - صيدليات (pharmacy)
  - مختبرات (lab)
  - مراكز أشعة (radiology)
  - عيادات أسنان (dental)
  - عيادات بصريات (optical)
  - علاج طبيعي (physiotherapy)
  - أخرى (other)

- **المعلومات المتاحة:**
  - معلومات أساسية (اسم، وصف، عنوان)
  - معلومات الموقع (منطقة، مدينة، إحداثيات)
  - معلومات الاتصال (هاتف، إيميل، موقع)
  - أنواع الخدمات الطبية (JSON)
  - الخصومات الطبية (JSON)
  - ساعات العمل (JSON)
  - التقييمات والإحصائيات

### 3. نظام المنشورات
- **تصنيفات المنشورات:**
  - أخبار طبية
  - نصائح صحية
  - خدمات طبية
  - عروض وخصومات
  - توعية صحية
  - أحداث ومؤتمرات

- **مميزات المنشورات:**
  - محتوى بالعربية والإنجليزية
  - مرفقات متعددة
  - حالات مختلفة (مسودة، منشور، مؤرشف، معلق)
  - أولويات مختلفة
  - تاجات وبيانات إضافية
  - إحصائيات المشاهدات والإعجابات

### 4. نظام التقييمات
- تقييم المراكز الطبية من 1 إلى 5 نجوم
- تعليقات مفصلة
- نظام موافقة للتقييمات
- حساب متوسط التقييمات تلقائياً

## هيكل قاعدة البيانات

### الجداول الرئيسية:
1. **users** - المستخدمين
2. **roles** - الأدوار
3. **permissions** - الصلاحيات
4. **role_user** - ربط المستخدمين بالأدوار
5. **permission_role** - ربط الأدوار بالصلاحيات
6. **medical_centers** - المراكز الطبية
7. **medical_center_reviews** - تقييمات المراكز الطبية
8. **post_categories** - تصنيفات المنشورات
9. **posts** - المنشورات
10. **post_attachments** - مرفقات المنشورات

## المسارات (Routes)

### المسارات العامة:
- `/` - الصفحة الرئيسية
- `/medical-centers` - قائمة المراكز الطبية
- `/medical-centers/{slug}` - تفاصيل مركز طبي
- `/posts` - قائمة المنشورات
- `/posts/{slug}` - تفاصيل منشور

### مسارات لوحة التحكم:
- `/admin` - لوحة التحكم الرئيسية
- `/admin/medical-centers` - إدارة المراكز الطبية
- `/admin/posts` - إدارة المنشورات
- `/admin/users` - إدارة المستخدمين
- `/admin/roles` - إدارة الأدوار
- `/admin/reviews` - إدارة التقييمات

### API Endpoints:
- `/api/medical-centers` - API المراكز الطبية
- `/api/posts` - API المنشورات
- `/api/categories` - API التصنيفات
- `/api/stats` - إحصائيات عامة
- `/api/admin/*` - API محمية للمديرين

## التشغيل

### متطلبات النظام:
- PHP 8.1+
- Laravel 11
- MySQL/PostgreSQL
- Composer

### خطوات التشغيل:
1. `composer install`
2. `cp .env.example .env`
3. `php artisan key:generate`
4. إعداد قاعدة البيانات في `.env`
5. `php artisan migrate:fresh --seed`
6. `php artisan serve`

### إعداد التخزين:
```bash
php artisan storage:link
```

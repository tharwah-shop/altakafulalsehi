## 🔗 المسارات المتاحة

### المسارات العامة:
- `/` - الصفحة الرئيسية
- `/medicalnetwork` - الشبكة الطبية
- `/regions` - المناطق
- `/region/{slug}` - تفاصيل منطقة
- `/city/{slug}` - تفاصيل مدينة
- `/medical-centers` - قائمة المراكز الطبية
- `/medical-center/{slug}` - تفاصيل مركز طبي
- `/posts` - المنشورات
- `/posts/{slug}` - تفاصيل منشور
- `/consultations/*` - الاستشارات الطبية
- `/offers` - العروض

### مسارات لوحة التحكم:
- `/admin` - لوحة التحكم الرئيسية
- `/admin/medical-centers` - إدارة المراكز الطبية
- `/admin/posts` - إدارة المنشورات
- `/admin/users` - إدارة المستخدمين
- `/admin/roles` - إدارة الأدوار
- `/admin/reviews` - إدارة التقييمات

### API Endpoints:
- `/api/medical-centers` - API المراكز الطبية
- `/api/posts` - API المنشورات
- `/api/categories` - API التصنيفات
- `/api/stats` - إحصائيات عامة

## 🗄️ قاعدة البيانات

### الجداول المنشأة:
1. **users** - المستخدمين (محدث)
2. **roles** - الأدوار
3. **permissions** - الصلاحيات
4. **role_user** - ربط المستخدمين بالأدوار
5. **permission_role** - ربط الأدوار بالصلاحيات
6. **medical_centers** - المراكز الطبية (مدمج مع المناطق والمدن)
7. **medical_center_reviews** - تقييمات المراكز الطبية
8. **post_categories** - تصنيفات المنشورات
9. **posts** - المنشورات
10. **post_attachments** - مرفقات المنشورات
